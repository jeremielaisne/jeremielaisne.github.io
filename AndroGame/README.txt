*Projet Unity 2018*
par laisné jérémie - IUT Orléans Licence Pro

Réalisé sur Unity : - 3 scènes compilé et testé sur Navigateur Firefox uniquement (en lançant l'index.html). Pas de test sur
			versions portable 

Le jeu : - Un petit shooter repris du TD, j'ai ajouté un singleton, du son, un décor dynamique, deux ennemis et le score.
	 - Il comprend 3 scènes (Un menu, le jeu, une scène de game-over).
	 - Un seul niveau (jeu de scoring)

Comment  y jouer : - Appuyer sur Start puis essayer de faire le meilleur score
		   - Haut/ Bas pour déplacer le vaisseau
	           - Touche entrée pour shooter

Règles : - -1 pts si on tir sur un asteroid
  	 - +1 pts si on touche un ennemi
	 - +4 pts toutes les 10 sec du jeu
	 - 5 vies avant un éventuelle GAME OVER (pas de limite de temps)

Sources :
	 - Documentation officiel Unity principalement (et forum d'entraide) 
	 - Le TD et cours de M.Martins
	 - Son : Mute City 8 Bit - F-Zero et Zelda WindWaker 16 bits(Bulbamike)

Difficultés : Les sprites et les backgrounds, je n'ai pas réussi à afficher un asteroid morcelé lorsque je le touche par exemple.
	Le déplacement des ennemis a été compliqué pour moi, j'ai eu du mal à le faire déplacer comme je le voulais.
	Tout le reste correspond un peu près ce que je voulais faire. 
	Mon plus gros problème a été la gestion du temps, j'aurais voulu ajouter 2 autres niveaux avec un boss final et une 
	arme supplémentaire mais avec les autres projets puis le stage je n'ai pas eu le temps de faire ce que je voulais faire.
